# llmzk-synthesize skill

Use to create wiki/article outputs from existing Zettelkasten notes.

Do not synthesize directly from raw source material. Synthesis uses the note graph: concept notes, permanent notes, bridge notes, contradiction notes, index notes, literature notes, and source trails.

For Obsidian syntax, wikilinks, frontmatter, embeds, and callouts, rely on the installed `obsidian-markdown` skill.

## Process

1. Read `AGENTS.md`, `.opencode/docs/SOUL.md`, and `.opencode/docs/OBSIDIAN_SKILLS.md`.
2. Identify the topic index or cluster.
3. Inspect concept, permanent, bridge, contradiction, and literature notes.
4. Produce an outline before writing the article.
5. Write the article in `08 Wiki Articles/`.
6. Include `derived_from` in frontmatter using quoted wikilinks.
7. Do not hide unresolved tensions; include them where useful.
8. Write or update a decision log.
